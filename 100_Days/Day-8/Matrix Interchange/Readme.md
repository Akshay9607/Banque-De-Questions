# Problem Description

The task is to swap the columns in a 2D array.<br>
You are given a matrix arr[][] of r rows and c columns. You need to swap the first column with the last column.

```py
Example:

Input:
r = 3
c = 4
arr[][] = [[1, 2, 3, 4],
           [4, 3, 2, 1],
           [6, 7, 8, 9]]

Output:
4 2 3 1
1 3 2 4
9 7 8 6
```

Print the modified matrix.

<br>

_Constraints:_
1 <= r, c <= 100
